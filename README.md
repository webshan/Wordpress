# Wordpress # 
This is a dockerized enviroment to WordPress developers.

## Base Usage ##
1- open `.env` file.  
2- set a unique value for `COMPOSE_PROJECT_NAME`.  
3- set WordPress version  
4- set MySQL version
5- set Port number that must be free to use.
6- run `docker-compose up -d`
